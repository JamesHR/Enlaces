# UTCV - Telecomunicaciones
Herramienta para cáalculo de enlaces, zona de fresnel y sistema fotovoltaico

- Framework: React

## Scripts
Instalación de modulos node
```bash
npm install
```
Ejecutar proyecto
```bash
npm start
```
